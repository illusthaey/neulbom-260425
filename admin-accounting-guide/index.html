8. 공문 작성, 결재선, 협조·공람

협조가 필요한 경우

  • 계약 체결 전 계약요구자료 제출
  • 징수·환불·지출 등 행정실 회계처리가 필요한 문서
  • 운영계획 변경, 강사 변경, 시설 이용 변경
  • 예산과목 또는 회계처리 가능 여부 검토가 필요한 경우

공람이 필요한 경우

  • 징수요구 결재 완료 후 행정실 담당자 확인
  • 과오납 반환 요구 결재 완료 후 행정실 확인
  • 강사료 지급 요구 결재 완료 후 지출담당자 확인
  • 외부 공문 처리 결과를 관련자가 알아야 하는 경우
작성 기본: 제목은 연도·월·업무명을 포함하고, 관련 근거는 운영계획·계약서·지침 순으로 쓰며, 붙임은 총괄표 → 세부명단 → 증빙 순으로 정리합니다.
