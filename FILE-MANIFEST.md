# 파일 구성표

## 루트

| 파일 | 용도 |
|---|---|
| `index.html` | 메인 페이지 |
| `404.html` | 오류 페이지 |
| `CNAME` | GitHub Pages 사용자 도메인 설정 |
| `CNAME.example` | CNAME 예시 |
| `robots.txt` | 검색엔진 크롤링 허용 |
| `sitemap.xml` | 실제 도메인 기준 사이트맵 |
| `sitemap.xml.example` | 도메인 교체용 사이트맵 예시 |
| `README.md` | 배포 및 수정 안내 |
| `FILE-MANIFEST.md` | 파일 구성표 |

## 정적 파일

| 경로 | 내용 |
|---|---|
| `static/style.css` | 전체 UI 스타일. 기존 디자인 시스템 + 늘봄 독립 사이트 보정 포함 |
| `static/site-config.js` | 사이트명, 메뉴, 페이지별 히어로, 목차, 푸터 설정 |
| `static/site.js` | 공통 헤더·히어로·목차·푸터 렌더링, 검색, 복사, 인쇄, 위로 버튼 |
| `static/alien.jpg` | 메인 히어로 이미지 |
| `static/favicon.ico` | 파비콘 |
